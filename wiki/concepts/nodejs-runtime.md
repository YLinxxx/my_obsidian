# Node.js 运行时

## 通俗总结

Node.js 是一个让 JavaScript 跳出浏览器运行的 runtime，特别常用于服务器、工具链和自动化脚本。

## 为什么重要

- 它让你能用 JavaScript 同时处理前端和后端
- 它非常适合 I/O 密集型服务
- 它是现代 JavaScript 工具生态的核心组成部分

## 关键理解

- runtime 就是执行代码的环境
- Node.js 提供了文件、网络、进程等 API
- non-blocking I/O 是它在后端场景里很重要的原因之一
