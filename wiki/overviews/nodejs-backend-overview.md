# Node.js 后端总览

## 总结

这页是当前 vault 中学习 Node.js 后端开发的顶层路线图。

Node.js 学习可以先按四层推进：

1. 运行时基础
2. 后端应用基础
3. 生产环境模式
4. 用项目进行巩固

## 核心板块

### 运行时基础

- [[wiki/concepts/nodejs-runtime|Node.js 运行时]]
- [[wiki/concepts/event-loop|事件循环]]
- 模块系统、包管理与依赖解析
- 基于 Promise 与 `async` / `await` 的异步编程
- streams、buffers 与文件 I/O

### 后端应用基础

- [[wiki/concepts/http-api-basics|HTTP API 基础]]
- 路由设计
- 中间件（middleware）
- 请求校验
- 数据库访问
- 认证与授权
- 环境配置

### 生产环境模式

- 日志
- 错误处理
- 测试
- 限流
- 缓存
- 后台任务
- 部署与可观测性

### 项目强化

- 写一个 REST API
- 加入数据库持久化
- 加入认证
- 部署一个小服务
- 补测试并改进错误处理

## 推荐学习节奏

对每个主题都尽量走完下面这几个动作：

1. 读一篇聚焦资料
2. 创建或更新对应的来源总结页
3. 把内容连接到某个概念页
4. 做一个很小的练习或例子
5. 记下还没弄懂的地方
