# AI 与 LLM 总览

## 总结

这页用于跟踪 AI 与 LLM 学习路线，它和 Node.js 学习线是互补关系。

目标不只是理解模型本身，还要理解怎样把模型做成真正可用的系统。

## 核心板块

- 模型基础
- prompt 与评测
- embeddings 与 retrieval
- agent 工作流
- tool use
- memory 与知识系统
- LLM 应用架构

## 为什么它对这个 Vault 很重要

这个 vault 本身就是一个 AI 辅助知识系统的实践案例。
[[wiki/concepts/llm-knowledge-wiki|LLM 知识 Wiki]] 既是学习工具，也是学习主题本身。

## 与 Node.js 的连接方式

随着后端能力提升，可以逐步把它们和 AI 应用连接起来：

- 围绕 LLM 能力封装 API
- 存储和检索结构化笔记
- 搭建 ingest pipeline
- 试验 agent 风格工作流
