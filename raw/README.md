# 原始资料说明

`raw/` 用来存放 wiki 的原始来源资料。

## 结构

- `raw/inbox/` 放新抓取但还没处理的资料
- `raw/sources/` 放你希望长期保留的来源记录或清理版资料

## 规则

- 把 raw 文件视为不可变资料
- 不要直接在 raw 里改写、压缩或“重写”原文
- 总结、提炼和结构化内容应写入 `wiki/`
- 如果资料很长，可以分段处理，并把结果页面互相链接起来

## 建议流程

1. 把新文章、笔记或 transcript 放进 `raw/inbox/`
2. 让 Codex 执行 ingest
3. 如有需要，把更适合归档的版本放入 `raw/sources/`
4. 确认 `wiki/`、`index.md` 和 `log.md` 已同步更新
